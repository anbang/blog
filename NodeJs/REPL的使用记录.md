语法:输入node命令，然后写下面
```
.save 保存一个文件夹
.save reademe.md 就可以新建一个reademe.md的文件；
```
然后在下面输入11+11 就可以在里面写入数据了；
```
.load 执行保存过的文件夹
.load reademe.md 可以执行11+11 返回22 
```
