`console`彩色字体，因为花哨，没啥用，而且一般都用于调试的，也没人搞那些；

因为经常F12下看网站，所以看到彩色的`console`，一下吸引了；

代码如下：
```javascript
console.log(
    "%c%s",
    "color: red; background: yellow; font-size: 24px; font-weight: bold;",
    "请不要使用console!"
)
```
没啥用，就是颜色变了下而已；


如果想稳妥，可以转uncode然后再放进去；
